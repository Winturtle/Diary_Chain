✅ `while` 的重要性

- **重複執行任務**：當你需要反覆檢查某個條件、等待輸入、持續監控感測器、或讓某個流程持續進行直到符合條件為止，`while` 迴圈就很有用。
- **控制流程**：它讓程式可以根據執行當下的狀態，動態地決定是否繼續迴圈，是邏輯與效率的重要工具。
- **資料處理常用**：像是讀取檔案到結尾、或搜尋清單中某個值，常用 `while` 迴圈來實作。
- 
迴圈示範遊戲
```// oldWin_P20250626_4 亂數選取遊戲

#include <iostream>
using namespace std;

int main()
{
	int quess, input_num, times = 0;

	srand(time(NULL));
	quess = rand();
	quess = quess % 99 + 1; //亂數產生1~99的數字
	
	while (times < 5) {
		cout << "Please input a number: ";
		cin >> input_num;
		if (input_num > quess) {
			cout << "Too Big!! \n";
		}
		else if (input_num < quess) {
			cout << "Too Small!! \n";
		}
		else { 
			cout << "恭喜你答對了 \n"; 
			break;
		}
		times++;
	}
	
	if (times >= 5) {
		cout << "你輸了! \n";
	}
}

```
do – while 敘述
```// oldWin_P20250626_5

#include <iostream>
using namespace std;

int main()
{
	int i, sum = 0;
	i = 0;
	do {
		sum += i;
		i++;
	} while (i <= 100);
	cout << "Sum = " << sum << "\n";

	i = 0; sum = 0;
	while (i <= 100) {
		sum += i;
		i++;
	}
	cout << "Sum = " << sum << "\n";
}

	int max;

int maximum(int x, int y, int z) { //define
	int temp;

	if (x >= y)
		temp = x;
	else
		temp = y;
	if (temp >= z)
		return(temp);
	else
		return(z);
}
```

建立函式如果在main之後，一定要到最上端進行宣告。
如果在main之前就不用宣告了。
